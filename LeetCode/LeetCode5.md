## 5. 最长回文子串

#### 问题描述

给定一个字符串 s，找到 s 中最长的回文子串。你可以假设 s 的最大长度为 1000。

示例 1：

```
输入: "babad"
输出: "bab"
注意: "aba" 也是一个有效答案。
```

示例 2：

```
输入: "cbbd"
输出: "bb"
```

#### 问题分析

使用中心扩散的思路，以每一个字符串为中心，向两边寻找最长的回文串。
